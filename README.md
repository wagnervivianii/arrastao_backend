Projeto Arrastão

Sistema desenvolvido para apoiar processos de validação, auditoria e acompanhamento de informações coletadas em campo.

O projeto tem como foco organizar dados operacionais, validar respostas, consultar evidências e permitir que a equipe responsável acompanhe inconsistências de forma mais rápida e estruturada.

Sobre o projeto

O Projeto Arrastão foi criado para melhorar o controle sobre dados de execução em campo, especialmente em cenários onde há grande volume de pesquisas, respostas, imagens e registros operacionais.

A aplicação centraliza informações relevantes para análise, permitindo visualizar dados consolidados, validar registros e apoiar decisões operacionais com mais segurança.

Principais funcionalidades
Consulta de registros coletados em campo
Validação de respostas operacionais
Organização de evidências, como imagens e informações complementares
Apoio ao processo de auditoria
Integração com banco de dados
Estrutura preparada para consumo por frontend
Endpoints para consulta, edição e registro de ações
Base para integrações com APIs externas
Tecnologias utilizadas
Python
Flask
SQLAlchemy
PostgreSQL
Pandas
APIs REST
Estrutura modular por rotas, serviços e consultas SQL
Objetivo técnico

O objetivo técnico do projeto é fornecer uma camada backend organizada para processar dados operacionais, consultar informações consolidadas no banco, aplicar regras de validação e entregar respostas estruturadas para a interface da aplicação.

O sistema foi pensado para evoluir em módulos, mantendo separação entre regras de negócio, rotas, consultas e integrações externas.
